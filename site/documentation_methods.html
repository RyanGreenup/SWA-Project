    MkDocs¶

    For the most part we can use MkDocs for documentation, and keep the documentation on the gh-pages branch of the repo.

    Setting it up¶

    It's super easy:

    1. Install python3
    2. Install MkDocs as described here
    3. I forget whether to do this through your package manager apt/pacman/brew... or through pip3, I think pip3 is the way to go.
    4. Install the Extensions as described here
    5. look in the yaml here to see what's been enabled thus far.
